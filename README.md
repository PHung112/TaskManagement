**Task Management System** là một ứng dụng web toàn diện giúp các team tối ưu hóa hiệu suất làm việc và quản lý công việc một cách khoa học. 
Hệ thống cung cấp giải pháp quản lý dự án với giao diện thân thiện và tính năng hợp tác nhóm mạnh mẽ.

### 🛠️ Công Nghệ Sử Dụng

**Frontend:**
- React 19.2.0 + Vite 
- Tailwind CSS v4 
- Axios 
- React Router DOM 
- WebSocket/STOMP

**Backend:**
- Java 17+
- Spring Boot 4.0.2
- Spring Data JPA (ORM)
- Spring Security (authentication & authorization)
- Maven (build tool)

### ✨ Các Chức Năng Chính

✅ **Tạo & Quản lý Tasks**
- Tạo công việc mới và assign cho team members
- Chỉnh sửa, cập nhật và xóa tasks
- Gán deadline và mô tả chi tiết cho từng task

📊 **Phân Loại Độ Ưu Tiên**
- Đánh dấu độ ưu tiên cho tasks (High, Medium, Low)
- Sắp xếp danh sách công việc theo độ khẩn cấp
- Giúp team focus trên những công việc quan trọng nhất

🔄 **Quản lý Trạng Thái**
- **To-do** (Chưa bắt đầu) - Task mới được tạo
- **Doing** (Đang làm) - Task đang được xử lý
- **Done** (Hoàn thành) - Task đã được hoàn tất
- Cập nhật trạng thái real-time giữa các team members

👥 **Quản lý Projects & Members**
- Tạo dự án và mời members tham gia
- Phân quyền chi tiết: ADMIN, MEMBER, VIEWER
- Quản lý vai trò và quyền hạn của từng user

📱 **Giao Diện Hiện Đại**
- Responsive design tối ưu cho desktop và mobile
- Sidebar dễ sử dụng để điều hướng
- Notification bell theo dõi cập nhật real-time

### 🎯 Why This Project?

Trong thời đại làm việc hiện đại, team cần công cụ giúp:
- **Tăng năng suất** - Không còn lẫn lộn với emails hoặc tin nhắn rời rạc, tất cả công việc được tập trung ở một nơi
- **Cải thiện hợp tác** - Real-time updates giúp team luôn đồng bộ thông tin
- **Dễ quản lý** - Interface trực quan giúp cả non-techy users cũng sử dụng dễ dàng
- **Khả năng mở rộng** - Kiến trúc API base cho phép tích hợp với các tools khác trong workflow
